# Requirements
- Node v18
- PostgreSQL v14

# Installation
1. Clone the repository.
   ```
   git clone git@github.com:minseo/mellow.git
   ```
2. Navigate to the project folder.
   ```
   cd mellow
   ```
3. Install dependencies.
   ```bash
   npm i
   ```

# Frontend Workflow
1. Navigate to the folder and start your VSC.
   ```bash
   code .
   ```

2. Start frontend server.
   ```bash
   npm run dev
   ```
